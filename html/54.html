单招考核方式

单招由学校自主命题，采取文化素质+职业技能考核方式，职业技能含职业素质和职业技能/适应性评价两部分；按高中生、中职生、无学考社会考生、退役士兵分四类考核，退役士兵免文化素质测试，各类考生计分规则不同。

一、各类考生考核明细

考生类型 文化素质测试(240分) 职业素质测试(130分) 职业技能/适应性评价(130分)
高中生(含学考中职生、社会考生) 语数外学业水平等级赋分
五档：语数A=100、B=90、C=70、D=40、E=10；外语A=40、B=36、C=28、D=16、E=4
四档：语数A=100、B=90、C=70、D=30；外语A=40、B=36		 50分钟闭卷笔试，考察专业素养、科学素质、职业潜能和倾向 分组面试，考察表达能力、创新思维、分析解决问题能力等综合素质
中职生 闭卷笔试，语文120分+数学120分，同场120分钟，依据中职公共基础课程标准命题 50分钟闭卷笔试，考察职业素养、职业潜能、职业倾向和综合能力 分组实操测试，依据中职专业简介命题，考查综合专业能力、岗位技能、通用技术等
社会考生(无学考) 闭卷笔试，语文120分+数学120分，同场120分钟，依据中职公共基础课程标准命题 50分钟闭卷笔试，考察职业素养、职业潜能、职业倾向和综合能力 分组面试，考察表达能力、创新思维、分析解决问题能力等综合素质
退役士兵 免考 50分钟闭卷笔试，考察职业素养、职业潜能、职业倾向和综合能力 随高中生编组面试，考察表达能力、创新思维、分析解决问题能力等综合素质

二、各类考生录取成绩计算方式

1. 高中生录取成绩 = 文化素质成绩 + 职业素质测试成绩 + 职业适应性评价成绩
2. 中职生录取成绩 = 文化素质测试成绩 + 职业素质测试成绩 + 职业技能评价成绩 + 赋分成绩
3. 社会考生录取成绩 = 文化素质测试成绩 + 职业素质测试成绩 + 职业适应性评价成绩 + 赋分成绩
4. 退役士兵考生录取成绩 = (职业素质测试成绩 + 职业适应性评价成绩) ÷ 0.52
